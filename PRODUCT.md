# Product

## Register

product

## Users

Wesley College teachers (PYP through Senior), planning lessons under time pressure. Not
technical users. They arrive with a stimulus (image, video, PDF, text, or link) and want a
classroom-ready deck in about two minutes. One shared school passcode, no sign-in.

## Product Purpose

Springboard turns a teacher-supplied stimulus into an editable 4-slide PowerPoint
(IGNITE → THINK → LAUNCH → REFLECT) built on a Project Zero Thinking Routine and bridged
to the teacher's curriculum. Success = stimulus to downloaded deck in ~2 minutes with no
training needed.

## Brand Personality

Playful, energetic, warm — a springboard, not a form. It should feel like a favourite
creative colleague, not an admin system. Bold friendly type, colour used with joy, small
moments of delight. Still trustworthy: Wesley College brand palette only, and the tool
must disappear into the task once a teacher is working.

Reference points (user-supplied): appwrite.io (premium modular polish, confident accents
on deep colour) and HeyFriends (bold type, sticker/hand-made playfulness, chunky rounded
shapes).

## Anti-references

- Grey enterprise admin panels; SharePoint-form energy.
- Generic SaaS cream-and-gradient template look.
- Anything that reads "compliance tool" rather than "creative tool".
- Playfulness that slows the task: no orchestrated load sequences, no decoration that
  competes with the Generate flow.

## Design Principles

1. **Two minutes, protected.** Delight lives in the margins; the stimulus → generate →
   download path stays obvious and fast.
2. **Wesley, worn lightly.** Purple is the only interactive voice; gold and the
   highlighter pastels carry the fun. No off-palette colour.
3. **Bounce in the shape, not the wait.** Personality comes from chunky radii, tactile
   buttons, stickers and colour — not from long animations.
4. **Teacher-first words.** Labels sound like a colleague ("Add a stimulus"), never like
   software ("Input source configuration").
5. **Every state dressed.** Empty, loading, error and success states get the same care as
   the happy path.

## Accessibility & Inclusion

WCAG 2.2 AA. All text-on-tint pairings use the AA-verified ink substitutes from the
Wesley brand kit. Solid high-contrast focus rings. Reduced-motion alternative for every
animation. Minimum 24×24 hit areas.
